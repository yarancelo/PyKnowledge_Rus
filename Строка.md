Строка в Python объявляется как
``` string = "something" ```
Вместо двойных кавычек возможно использовать одинарные, а для мультистроковой строки можно использовать тройные кавычки.

Длину строки можно измерить с помощью [[Встроенные функции#^4de148|len()]].
[[Встроенные функции#^ca6a5e|bool()]] от любой непустой строки будет равна True
Строки можно конкатенировать с помощью оператора +, например, ```"a"+"b"="ab"```
Строки можно умножать с помощью оператора умножения, например, ```"a"*3="aaa"```
Если попробовать узнать тип строки через [[Встроенные функции#^583cd1|type()]], то, ответом будет ```<class 'str'>```

Из строки можно получить целое число с помощью [[Ключевое слово int()]], а также можно получить вещественное число с помощью [[Ключевое слово float()]].
Строки можно сравнивать используя [[Операторы сравнения]].

Для того, чтобы найти вхождения подстроки в строку можно использовать [[Оператор in]].

**Форматирование строк**
В строки можно подставлять значения используя "f-строки"
~~~
name = "Vova"
age = 22
hello_str = f"Hello, my name is {name} and i'm {age} years old"
print(hello_str)

В консоль будет выведено "Hello, my name is Vova and i'm 22 years old"
~~~

Также, можно форматировать строки, используя %
~~~
name = "Vova"
age = 22
hello_str = f"Hello, my name is %s and i'm %d years old" % (name, age)
print(hello_str)

В консоль будет выведено "Hello, my name is Vova and i'm 22 years old"
~~~
Данный способ форматирования строк был позаимствован из языка C. 
Индексацию и итерирование по строкам можно осуществлять согласно правил [[Индексация и слайсы|индексации]].

#### **[[Метод|Методы]] строк:**
* Метод str.upper() - Превращает все строчные буквы в строке в заглавные.
* Метод str.lower() - Превращает все заглавные буквы в строке в строчные.
* Метод str.strip() - Убирает заданные символы по обоим краям строки.
* Метод str.lstrip() - Убирает заданные символы с левого конца строки.
* Метод str.rstrip() - Убирает заданные символы с правого конца строки.
* Метод str.replace() - Заменяет все вхождения одной подстроки на другую.
* Метод str.capitalize() - Превращает первую букву строки в заглавную.
* Метод str.title() - Превращает первую букву каждого слова в заглавную.
* Метод str.swapcase() - Меняет регистр каждой буквы в строке на строки на противоположный.
* Метод str.isdigit() - Проверяет, является ли строка целым числом.
* Метод str.count() - Подсчитывает количество вхождений одной подстроки в другую.
* Метод str.split(<разделитель>) - Возвращает [[Список]] подстрок, разделенных разделителем, переданным как аргумент.
