Условный оператор if имеет следующую структуру:
~~~
if <условие>:
	действие
~~~

В случае, когда нужен else:
~~~
if <условие>:
	действие
else
	действие
~~~

Если же нам нужно три условия, то мы можем воспользоваться следующей конструкцией:
~~~
if <условие>:
	действие
elif <условие>
	действие
else
	действие
~~~
